# FitTrack - Fitness and Task Management Dashboard

FitTrack is a modern, responsive web application designed to help users manage their fitness journey and daily tasks in one place. Built with React, Material-UI, and Vite, it provides a clean, intuitive interface for tracking workouts, monitoring health metrics, and staying organized.

## ğŸš€ Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/KubanjaElijahEldred/To-Do-List-App.git
   cd To-Do-List-App/frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn
   ```

3. Start the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.
